## Agent Coordination Protocol

Agents do NOT trigger each other directly (prevents bot storms and runaway costs).

**Flow:**
1. Agent completes task → writes output to brain/ folder
2. Agent posts to Slack channel
3. Morning Brief reads all outputs during daily brief
4. Curtis reviews and dispatches follow-ups

**Exceptions:**
- Cost-Sentinel can alert immediately for cost threshold violations
- Sentinel can alert immediately for security events
- Watchman can broadcast Sabbath activation to all agents
- Synthesizer can trigger Advisory Council members
