# J5 Slack Channel Guide
**Last Updated:** 2026-03-03
**Rule:** You never need to remember anything. J5 handles routing. This doc explains what each channel is for if you're curious.

---

## ⚡ J5 SYSTEM

### #j5-command-center
**What it is:** High-priority alerts and important J5 updates only. The one channel you should actually check.
**What shows up here:** Security alerts, cost threshold breaches, critical system failures, anything Sentinel flags as urgent.
**Your job:** Just read and respond if action needed. Everything else is informational.

### #j5-morning-brief
**What it is:** Your daily brief, delivered here every morning at 5:30 AM (Tue–Sun).
**What shows up here:** Today's calendar, tomorrow's preview, top Todoist tasks, security snapshot, cost snapshot, pastoral word.
**Your job:** Read it at 4:30 AM. React with ✅ when you've seen it. Nothing else required.

### #j5-sentinel
**What it is:** Security and system health reports.
**What shows up here:** Nightly security audit results, firewall status, exposed credential warnings, cron health checks.
**Your job:** Review weekly. Act on anything flagged 🔴. Yellows are informational.

### #j5-cost-tracker
**What it is:** Daily API spend and monthly budget tracking.
**What shows up here:** Daily cost alert (9 AM), weekly spend summaries, threshold breach alerts.
**Your job:** Glance daily. Alert if over $10/day. Monthly budget target: under $50 for Claude Code.

### #j5-system
**What it is:** Infrastructure updates — crons, builds, deployments, git pushes.
**What shows up here:** Cron completions, build notifications, git sync status, DB backup confirmations.
**Your job:** Ignore unless something is red. This is J5's engine room log.

## 🤖 J5 AGENTS

### #j5-shepherd
**What it is:** Relationship health reports and touchpoint reminders.
**What shows up here:** Weekly relationship check (Thursdays 8 AM) — who needs attention, last contact dates, any 🔴 red flags.
**Your job:** Thursday morning: review and act. If someone is red, reach out that day.

### #j5-scribe
**What it is:** Meeting prep and post-meeting summaries.
**What shows up here:** Pre-meeting briefs (when you run /intake for a meeting), post-meeting summaries with action items extracted.
**Your job:** Before a meeting: check here. After a meeting: your tasks are already in Todoist.

### #j5-horizon
**What it is:** Idea analysis and weekly synthesis reports.
**What shows up here:** Every /intake analysis (3-line summary), Sunday 7 PM weekly synthesis.
**Your job:** Sunday evening: read the weekly synthesis. It tells you what your week's inputs are pointing toward.

### #j5-ff-catalyst
**What it is:** Flawed & Flourishing content ideas and Catalyst agent outputs.
**What shows up here:** Product ideas, content angles, F&F strategy proposals, market signals.
**Your job:** Review weekly. Flag anything worth pursuing with a 🔥 reaction.

---

## HOW TO USE SLACK WITH J5

**You don't need to post in J5 channels.** J5 posts. You read and react.
**Commands stay in Telegram.** `/intake`, `/todoist`, `/shepherd` — all in Telegram, never in Slack.
**React to acknowledge.** ✅ = seen it. 🔥 = pursue this. ❓ = need more info.
**Slack is output. Telegram is input.** That's the whole model.
