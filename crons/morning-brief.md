# Morning Brief Cron
# Fires: 4:30 AM CST daily (except Monday - Sabbath)
# Model: claude-haiku-4 (cost-efficient)
# Output: Telegram message to Curtis

## What it delivers:
1. Today's date + day of week
2. Identity stack reminder (which role leads today based on calendar)
3. Top 3 Todoist tasks for today
4. Any calendar events (once Google Calendar connected)
5. Cost report: yesterday's API spend
6. Security: any flags from overnight
7. One grounding word — Scripture or formation prompt

## Sabbath rule: Monday = NO BRIEF. Silence.
## Sacred time rule: Never fires between 6:05–6:50 AM

## Cost estimate: ~$0.02 per brief (Haiku model)
