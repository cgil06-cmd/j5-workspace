## Summary
Cody Schneider (growth marketing expert) joins the podcast to live-demo how to use AI agents (Claude Code), MCPs, and a stack of marketing tools to automate customer acquisition 24/7. He calls this "GTM engineering" — originally coined by clay.com, now evolving into autonomous marketing via agent swarms.

## Key Concepts
- **GTM Engineering:** Using AI agents to do all "middle work" (everything that touches a keyboard) so humans only set direction and polish output
- **Agent Jockeying:** Running 10+ Claude Code instances simultaneously across different tasks
- **Autonomous Marketing Loop:** Ideate → Bulk create → Analyze → Optimize → Deploy (all automated)
- **Domain vocabulary as superpower:** The better your professional vocabulary, the better output you get from AI agents
- **API-first mindset:** APIs are now the product; UIs are optional. Every company will become an API company (Sam Altman)
